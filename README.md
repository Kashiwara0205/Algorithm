# algorithm
There are some algorithm made by nim in this repository.

## Usage
If Install nim and you execute following command, You can get executable files.

```
make compile
```

## implemented algorithm list

### sort
- bubble_sort
- bucket_sort
- counting_sort
- hash-based_search
- heap_sort
- insertion_srot
- quick_sort

### search
- binary_search
- breadth-first_search
- depath-first_search
- hash-based_search
- sequential_search
- string-hash-based_search

### graph

- dijstra_dg

- floyd-warshall
